# Permit-web-app

# Название: 
Web приложение для автоматизации системы оформления и выдачи пропусков для автотранспорта

## Состав проекта:
- база данных на MSSQL
- бэкенд выполняется на языке C#
- фронтэнд выполняется на ViewJS
